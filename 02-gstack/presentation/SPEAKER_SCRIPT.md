## Slide 2 — gstack là gì? (~45 giây)

"**gstack** là virtual engineering team — bộ skills như nhân viên thật:

- `/office-hours` — CEO scope MVP
- `/design-consultation` — design system
- `/plan-eng-review` — khóa API và database
- `/qa` — test browser + screenshot

Không phải framework thay thế — là cách orchestrate AI agent có quy trình."

## Slide 3 — Custom gstack (~60 giây) — TRỌNG TÂM

"Team **custom** thêm **Role Council**:

1. **BA** phân tích đề → `BA_BRIEF.md` (user stories, acceptance criteria)
2. **Designer** search reference → `DESIGN.md` responsive
3. **Eng** lock API SQLite + docs đầy đủ
4. Build web → QA 375px → **hỏi có gen Flutter** cùng API

Điểm khác: không code trước phân tích; web responsive bắt buộc; demo account sẵn nếu có login."
