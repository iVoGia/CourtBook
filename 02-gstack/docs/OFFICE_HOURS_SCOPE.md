# Design: Sport Court Booking — 60-Minute Battle MVP

Generated by /office-hours on 2026-06-26  
Mode: Builder (hackathon fast-track)  
Status: DRAFT

## Problem Statement

Người chơi thể thao (cầu lông, mini football, tennis) cần đặt sân nhanh trên điện thoại — tránh gọi điện, tránh đến trực tiếp mà không biết slot còn trống. Pain: mất thời gian, double-booking, không theo dõi được lịch đã đặt.

## What Makes This Cool

Đặt sân trong **3 bước**: chọn sân → chọn ngày/giờ → xác nhận — thấy giá realtime, lịch cá nhân upcoming/past, hủy có cảnh báo 24h. Mobile-first, cảm giác app thật (không mock JSON).

## Constraints

- **60 phút** — cắt MVP theo MASTER_PROMPT
- Stack cố định: Vite React + Express + SQLite
- Responsive bắt buộc: 375px + desktop
- Auth demo + seed users
- Không payment gateway thật

## Premises

1. **Wedge hẹp đủ demo:** auth + danh sách sân + booking flow + my bookings là core loop — phải chạy end-to-end trên web responsive.
2. **Calendar tuần / recurring / QR / admin đầy đủ** là stretch — chỉ sau khi web QA pass.
3. **SQLite persistence** chứng minh data không mất khi refresh.
4. **User chính** là người chơi (role User); Admin chỉ cần seed courts + optional login demo.

## Recommended Approach (Approach A — Booking Loop First)

| Layer | 60-min IN | Stretch OUT |
|-------|-----------|-------------|
| Auth | Login/register demo, User + Admin roles | — |
| Courts | List 3 seeded courts (badminton, football, tennis) | Admin CRUD UI |
| Booking | Select court → date → slot → form → confirm + price calc | — |
| My Bookings | Upcoming / past, status, cancel + 24h warning | — |
| Calendar | Slot availability trong booking flow | Weekly calendar view |
| Business rules | 2h advance, 1–3h duration, capacity, no overlap | Operating hours config UI |
| Bonus | — | Recurring, QR, dashboard, mock payment |

## Build Order (sau design gate)

1. Scaffold → db:init + seed (users + 3 courts)
2. API: auth, courts, bookings CRUD
3. UI: login → courts → book → my bookings (responsive)
4. QA 375px + desktop → Flutter gate → PPTX
