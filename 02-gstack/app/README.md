# Full-Stack Contest Template

Monorepo: Vite React client + Express SQLite server.

```bash
npm install
npm run db:init
npm run dev
```

| Service | URL |
|---------|-----|
| Web | http://localhost:5173 |
| API | http://localhost:3001 |

**Responsive:** `client/` includes sidebar (desktop) + bottom nav (mobile 375px).

Copy: `bash ../scripts/new-fullstack-project.sh app`

Optional Flutter: after web QA → `bash ../scripts/scaffold-flutter-mobile.sh app`
