# GStack Role Council + Full-Stack + Flutter Gate

> Custom layer trên gstack — BA + Designer kỹ, web responsive, Flutter optional, PPTX 3 phần.

## Vì sao custom?

gstack factory đầy đủ nhưng 60 phút cần:
1. **Phân tích đề kỹ** (BA trước design)
2. **Design đẹp có research** (Designer + WebSearch/browse)
3. **Web responsive** + full-stack thật
4. **Flutter gate** — user chọn sau web OK
5. **PPTX** tách rõ: sản phẩm / gstack / custom

## Pipeline

```
office-hours → BA_BRIEF → design-consultation (DESIGN.md)
  → plan-eng-review → plan-design-review → docs/
  → scaffold → build web → QA responsive
  → [Flutter gate] → PPTX
```
