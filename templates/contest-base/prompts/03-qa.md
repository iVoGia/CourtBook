# Prompt 03 — QA (phút 45–50)

```
Chạy QA checklist trước khi làm PPTX:

1. npm run dev — app mở được (ghi URL)
2. Click từng nav link — không 404, không dead
3. Tạo/sửa/xóa 1 record end-to-end (hoặc flow chính của đề)
4. Refresh page — data còn (nếu có persistence)
5. Mobile width 375px — layout không vỡ
6. Chụp 3 screenshot lưu vào presentation/screenshots/:
   - 01-dashboard.png (hoặc màn chính)
   - 02-feature.png (tính năng core)
   - 03-result.png (kết quả / báo cáo)
7. Fix mọi dead button trước khi làm PPTX

Báo cáo ngắn: pass/fail từng mục. Fix lỗi critical ngay.
```
