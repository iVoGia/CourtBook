### GSD — Get Shit Done

| Phút | Bước | Prompt |
|------|------|--------|
| 2–8 | New project | `gsd-new-project` + paste đề |
| 8–12 | Discuss | **Chỉ 1 phase** — không multi-phase |
| 12–15 | Plan | Plan phase duy nhất |
| 15–42 | Execute | Execute parallel tasks trong phase |
| 42–45 | Verify | Verify deliverables |
| 50–57 | Presentation | `prompts/04-presentation-agent.md` |

**Quan trọng:** 60 phút chỉ đủ 1 phase — đừng over-scope.
